A **modern, interactive, and fully responsive React dashboard** dedicated to the legendary cricketer **Mahendra Singh Dhoni**.

This project showcases Dhoni’s career, achievements, stats, and legacy with a **premium UI, smooth animations, and theme switching experience**.

## 📌 Project Overview

The **MS Dhoni Dashboard** is a fan-made web application built using **React + Vite** that presents:

- Career highlights
- Achievements & milestones
- Detailed statistics
- ICC & IPL trophies
- Personal life insights
- Fan zone with quotes & iconic moments

All displayed in a **clean, modern, glassmorphism-based UI**.

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/ms-dhoni-dashboard.git
```

### 2️⃣ Navigate to project

```bash
cd ms-dhoni-dashboard
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run development server

```bash
npm run dev
```
